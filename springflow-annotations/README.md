# SpringFlow Annotations

Core annotation definitions for the SpringFlow framework.

## Annotations

This module will contain:
- `@AutoApi` - Main annotation to auto-generate REST API
- `@Filterable` - Configure field-level filtering
- `@Hidden` - Exclude fields from DTOs
- `@ReadOnly` - Mark fields as read-only
- `@SoftDelete` - Enable soft delete functionality (Phase 2)
- `@Auditable` - Enable audit trail (Phase 2)

## Dependencies

This module has zero external dependencies to remain lightweight.
